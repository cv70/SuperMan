# SuperMan AI Multi-Agent System - Frontend

## 前端开发指南

本文档说明了SuperMan多智能体系统的前端界面设计和开发。

### 界面特性

#### 1. 中文界面
所有界面文本均采用中文，包括：
- 标题和导航栏
- 数据卡片标签
- 按钮和操作项
- Agent角色名称
- 任务状态和优先级描述

#### 2. 现代化设计
- 使用深色主题 (#0a192f背景)
- 渐变色Accent (#2cc8ff到#66fcf1)
- 圆角卡片设计
- 平滑过渡动画
- 悬浮阴影效果

#### 3. 主要组件
- **系统概览**: 活跃智能体数量、已完成/待处理任务、紧急任务数
- **智能体健康状态**: 展示所有Agent的健康度、任务数、最后活动
- **任务优先级分布**: 可视化饼图展示任务优先级分布
- **正在进行的任务**: 表格形式展示所有任务的详细信息
- **系统活动流**: 实时消息流显示Agent间通信
- **智能体目录**: 平铺展示所有Agent卡片

### 技术栈
- **Vue 3**: 响应式框架
- **TypeScript**: 类型安全
- **Tailwind CSS**: 原子化CSS框架
- **Vite**: 构建工具

### 开发命令
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动预览
npm run preview
```

### 后端API集成
前端通过API服务与后端通信，主要接口：
- `GET /api/agents`: 获取Agent列表
- `GET /api/tasks`: 获取任务列表
- `GET /api/messages`: 获取消息列表
- `POST /api/send`: 发送消息
- `GET /api/status`: 获取系统状态

### 设计规范

#### 配色方案
- 主背景: `#0a192f` (深海军蓝)
- 次背景: `#112240` (更暗)
- 边框: `#233554`
- 强调色: `#2cc8ff` (青色)
- 成功率: `#66fcf1` (霓虹青)
- 警告色: `#ff6b6b` (红色)

#### 状态颜色
- **健康**: 绿色渐变
- **忙碌**: 黄色
- **空闲**: 灰色
- **错误**: 红色

#### 优先级颜色
- **紧急**: 琥珀色
- **重要**: 红色
- **一般**: 蓝色
- **普通**: 绿色
